## 📦 安装

组件库处于typescript迁移阶段，请勿使用

```bash
npm install zhui
```

## 🔨 使用

```jsx
import { Button } from 'zhui';
ReactDOM.render(<Button />, mountNode);
```

## ⌨️ 开发

clone locally:

```bash
$ git clone git@github.com:zhui-team/zhui.git
$ cd zhui
$ npm install
$ npm run ddev or npm run storybook
```
